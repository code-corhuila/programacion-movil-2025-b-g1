# Scrum

## Roles de Scrum

1. *Product Owner(PO)*
  - Representa a cliente/stakeholders
  - Define y prioriza el *Backlog del producto*
  - Asegura que el equipo entregue valor al negocio

2. *Scrum Master(SM)*
  - Facilita el proceso Scrum
  - Elimina impedimentos que bloquean al equipo
  - Asegura que se sigan los principios agiles

3. *Equipo de desarrollo (dev team)*
  - Autoorganizado y multifuncional
  - Define como se hara el trabajo 
  - Entrega el incremento de producto al final de cada Sprint
