# FadeApp – Agenda Digital para Barberías  

Este proyecto fue desarrollado en **Ionic React** como parte del taller de construcción de layouts (MosCow – Priorización).  

## 📌 Descripción
FadeApp es una aplicación móvil que permite a los usuarios:
- Visualizar barberías disponibles.  
- Seleccionar una barbería.  
- Agendar citas eligiendo fecha y hora.
